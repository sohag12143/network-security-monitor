# network-security-monitor🛡️ Network Security Monitor
A Python-based lightweight tool for monitoring network status, detecting open ports, and checking host availability.

---

🚀 Project Overview
This tool helps analyze basic network behavior and security exposure of a target system. It focuses on foundational cybersecurity concepts like connectivity testing and port monitoring.

It is designed for learning and demonstrating practical networking + Python automation skills.

---

✨ Key Features
🔍 Host Reachability Check – Verifies if a target system is online  
🌐 Port Detection – Scans common open ports  
⚡ Lightweight Execution – Fast and simple script  
🖥️ CLI Tool – Easy command-line usage  
🧠 Beginner-Friendly Code – Clean and understandable structure  

---

🧠 Skills Demonstrated
- Python automation
- Network socket programming
- Basic cybersecurity concepts
- Command-line tool development
- System connectivity analysis

---

🛠️ Tech Stack
- Python 3
- socket (built-in)
- sys module
